# Quickstart: fenix-6-support Validation

1. Build the app with Connect IQ SDK (recommend 4.1.6+): monkeyc & monkeydo configured
2. Install the .prg on a physical fenix 6 device or use the simulator profile for fenix 6
3. Run the app and start a match; exercise primary flows (start/stop, try, conversion, sin-bin)
4. Let the app run for 30 minutes and capture FIT session output
5. Verify timers remain synchronized and haptic patterns occur at expected thresholds
6. Upload FIT file to Garmin Connect to validate activity type and data integrity
