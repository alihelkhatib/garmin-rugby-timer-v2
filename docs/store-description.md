# Description

Rugby Timer is a match-day watch app for rugby referees who need clear timing and quick event controls on the wrist.

Run the match clock, pause and resume play, track each half, and record common scoring events for Home and Away teams. The main screen keeps the countdown, elapsed time, half, score, try counts, card status, and conversion timer visible during the match.

Features include:

- Rugby 15s, 7s, 10s, and U19 timing presets
- Pre-match half-length adjustment
- Start, pause, resume, end-match, and reset controls
- Try, conversion, penalty goal, and drop goal scoring
- Automatic conversion countdown after a try
- Yellow-card countdowns and red-card indicators
- Haptic reminders for paused match time and key timer alerts
- Half-time transition support
- Post-match event summary
- Garmin activity recording where supported by the device

Built for simple in-match use: quick buttons, readable match state, and no account or network dependency.
