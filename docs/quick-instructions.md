# Quick Instructions

1. Before kickoff, use `Up` and `Down` to adjust the half length, and `Menu` to choose a rugby variant.
2. Press `Select` to start the match, then use `Select` again to pause or resume.
3. Use the scoring and card menus during a running match to record tries, conversions, penalty goals, drop goals, yellow cards, and red cards.
4. A try starts the conversion countdown automatically.
5. When the countdown reaches zero, the half ends automatically.
6. Use `Back` during an active or completed match to end or reset the match.
