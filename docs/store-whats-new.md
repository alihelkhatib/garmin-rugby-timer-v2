# What's New

Initial release of Rugby Timer.

- Added rugby match timing with start, pause, resume, half-time, and match-end flow
- Added 15s, 7s, 10s, and U19 presets
- Added Home/Away scoring for tries, conversions, penalty goals, and drop goals
- Added automatic conversion timer after tries
- Added yellow-card countdowns and red-card indicators
- Added paused-time haptic reminders and timer alerts
- Added post-match event summary
- Added Garmin activity recording support on compatible devices
